Getting Started with AbhiStat

Your comprehensive guide to accessing and navigating the statistical analysis platform

1. Authentication and Account Creation

1.1 Understanding User Types

AbhiStat supports two types of users with different authentication methods:

Internal Abhitech Users

  • Employees of Abhitech Energycon Limited
  • Use company email credentials
  • Direct access through internal authentication system
  • Access via the main login page

External Users

  • Clients, partners, and external collaborators
  • Require account registration
  • Support for Google OAuth login
  • Access via external user login page

1.2 Abhitech Employee Login Process

1

Access the Login Page

Navigate to the AbhiStat application. You'll be presented with the main login interface featuring the Abhitech branding.

URL: https://abhistat.com/login
2

Enter Your Credentials

Use your Abhitech company email address and password:

  • Email: Your @abhitechenergycon.com email address
  • Password: Your company account password
Tip: Use the eye icon to toggle password visibility if needed.
3

Click "Log In As Abhitech Employee"

The blue gradient button will authenticate you through the internal system and redirect you to the data analysis workflow.

4

Successful Login

Upon successful authentication, you'll be automatically redirected to the Data File Checks page to begin your analysis.

1.3 External User Registration Process

1

Access External User Registration

From the main login page, click "Login as external user" to access the external authentication system.

2

Navigate to Sign Up

On the external login page, click "Sign Up" to create a new account.

3

Complete Registration Form

Fill out all required fields:

  • Name: Your full name
  • Email: Valid email address (will be your username)
  • Phone Number: Contact number
  • Password: Secure password
  • Confirm Password: Re-enter password for verification
4

Submit Registration

Click "Sign Up" to create your account. You'll be redirected to the login page upon successful registration.

1.4 External User Login Process

1

Standard Login

Enter your registered email and password, then click "Login".

2

Google OAuth Login (Alternative)

Click the Google Sign-In button to authenticate using your Google account. This creates an account automatically if you're a first-time user.

Note: Google OAuth requires proper configuration and may not be available in all environments.

1.5 Authentication Troubleshooting

Common Login Issues

Invalid Email or Password

Symptoms: Error message "Invalid email or password"

Solutions:

  • Verify your email address is correct
  • Check for typos in your password
  • Ensure caps lock is not enabled
  • For Abhitech employees, contact IT support for password reset
Account Not Found

Symptoms: Login fails for external users

Solutions:

  • Verify you're using the external user login page
  • Check if you need to register first
  • Ensure you're using the correct email address
Google OAuth Issues

Symptoms: "Google Sign-In failed" error

Solutions:

  • Ensure pop-ups are enabled in your browser
  • Clear browser cache and cookies
  • Try using standard email/password login instead
  • Contact technical support if the issue persists
Server Connection Issues

Symptoms: "Server error. Please try again." message

Solutions:

  • Check your internet connection
  • Wait a few minutes and try again
  • Clear browser cache
  • Contact technical support if the problem continues

2. Platform Overview and Navigation

2.1 Four-Step Analysis Workflow

AbhiStat follows a systematic four-step process for statistical analysis:

1

Data File Checks

Upload and validate your 'before' and 'after' data files

  • File format validation (CSV, Excel, Parquet)
  • Data structure verification
  • Column matching between datasets
→
2

Calculated Columns

Create custom formulas and derived metrics

  • Excel-like formula builder
  • Mathematical operations
  • Column references and functions
→
3

Dependency Model

Define variable relationships for analysis

  • Drag-and-drop variable selection
  • Independent vs dependent variables
  • Statistical significance testing
→
4

Data Visualization

Generate charts and statistical insights

  • Distribution curves
  • Scatter plots
  • Bootstrapping analysis

2.2 Key Platform Features

Multi-Format File Support

Upload data in CSV, Excel (.xlsx, .xls), or Parquet formats with automatic format detection and validation.

Interactive Formula Builder

Create calculated columns using an intuitive interface with real-time validation and preview capabilities.

Statistical Analysis Engine

Perform robust statistical comparisons with confidence intervals, significance testing, and bootstrapping.

Dynamic Visualizations

Generate interactive charts including distribution curves, scatter plots, and multi-variate analysis.

Session Management

Automatic data persistence during your analysis session with secure temporary file handling.

Export Capabilities

Download results, charts, and processed data for reporting and further analysis.

2.3 User Interface Navigation

Top Navigation Bar

  • Logo: Click to return to the landing page
  • User Menu: Access account settings and logout options
  • Help: Quick access to documentation and support

Step Navigation

  • Previous/Next Buttons: Navigate between workflow steps
  • Step Indicators: Visual progress through the four-step process
  • Breadcrumbs: Show current location in the workflow

Data Management Controls

  • Clear Data Button: Reset current session and start over
  • File Upload Areas: Drag-and-drop or click to select files
  • Preview Tables: Scrollable data previews with column headers

Interactive Elements

  • Formula Builder: Point-and-click interface for creating calculations
  • Variable Selector: Drag-and-drop interface for dependency modeling
  • Chart Controls: Customization options for visualizations
