Data Visualization

Create powerful statistical visualizations to analyze and interpret your data relationships and distributions.

Overview: The Data Visualization module is the fourth and final step in the AbhiStat workflow. After completing dependency modeling, you can create various types of charts and plots to visualize your statistical analysis results.

5.1 Distribution Curves

Distribution curves help you visualize the frequency distribution of your data, comparing patterns between "with product" and "without product" conditions.

Accessing Distribution Curves

  1. Complete the dependency modeling step
  2. Navigate to the "Visualize Data" page
  3. Select the "Distribution Curves" tab (first tab)

Creating Distribution Curves

Step 1: Column Selection

  1. Use the Column Selection dropdown to choose the variable you want to analyze
  2. The dropdown contains all available numeric columns from your dataset
  3. The chart will automatically update when you select a different column
Available Column Types:
  • Numeric columns from your original dataset
  • Calculated columns created in the previous step
  • Dependent and independent variables from dependency modeling

Step 2: View Mode Selection

Choose how to display your distribution data:

  • Combined View: Shows both datasets on the same chart for easy comparison
  • Separate Views: Displays "with product" and "without product" data in separate charts
  • Single View: Shows only one dataset at a time

Step 3: Chart Customization

Click the Settings icon to customize your chart:

  • Legend Labels: Customize the names displayed in the chart legend
  • Axis Labels: Set custom X and Y axis labels
  • Colors: Choose different colors for each dataset
  • Bin Count: Adjust the number of bins for the histogram (default: 20)
  • Grid Lines: Toggle grid lines on/off
  • Area Opacity: Adjust the transparency of the distribution curves

Understanding Distribution Curves

Chart Elements

  • X-Axis: Shows the range of values for the selected variable
  • Y-Axis: Shows the frequency percentage for each bin
  • Colored Areas: Represent the distribution shape for each dataset
  • Legend: Identifies which color represents which dataset

Statistical Insights Panel

Click the Distribution Analysis panel to view detailed statistics:

  • Mean: Average value for each dataset
  • Standard Deviation: Measure of data spread
  • Skewness: Indicates if the distribution is symmetric or skewed
  • Data Quality Score: Overall assessment of data completeness and outliers

Distribution Summary Cards

The summary cards provide quick insights:

  • Total Data Points: Number of observations in each dataset
  • Distribution Shape: Symmetric, right-skewed, or left-skewed
  • Tail Behavior: Normal, heavy-tailed, or light-tailed
  • Data Quality: Score from 0-100 based on completeness and outliers

Interpreting Distribution Curves

Shape Analysis

  • Normal Distribution: Bell-shaped curve, symmetric around the mean
  • Right-Skewed: Tail extends to the right, most data on the left
  • Left-Skewed: Tail extends to the left, most data on the right
  • Bimodal: Two peaks, indicating two different populations

Comparison Analysis

  • Similar Shapes: Suggests consistent behavior between conditions
  • Different Means: Indicates a shift in central tendency
  • Different Spreads: Shows variation in data consistency
  • Different Shapes: Suggests fundamental differences in behavior

Export and Sharing

  • Download PNG: Click the download icon to save charts as high-resolution images
  • Watermark: All exported charts include the AbhiStat watermark
  • File Naming: Charts are automatically named with the variable and chart type

5.2 Scatter Plots

Scatter plots visualize the relationship between two continuous variables, helping you identify correlations, trends, and outliers.

Creating Scatter Plots

Step 1: Variable Selection

  1. Navigate to the Scatter Plot tab
  2. Select the Independent Variable (X-axis) from the dropdown
  3. Select the Dependent Variable (Y-axis) from the dropdown
  4. Variables are automatically populated from your dependency modeling results

Step 2: Dataset View Selection

Choose which datasets to display:

  • Both: Shows both "with product" and "without product" data
  • With Product: Shows only the "with product" dataset
  • Without Product: Shows only the "without product" dataset

Step 3: Chart Customization

Use the settings panel to customize your scatter plot:

  • Point Size: Adjust the size of data points (4-16 pixels)
  • Opacity: Control point transparency (30-100%)
  • Colors: Set different colors for each dataset
  • Trend Lines: Toggle linear regression lines on/off
  • Grid Lines: Show/hide background grid
  • Outliers: Highlight outlier points in orange

Interactive Features

Zoom and Pan

  • Mouse Wheel: Zoom in and out of the plot
  • Click and Drag: Pan around the zoomed plot
  • Zoom Controls: Use the zoom in/out buttons
  • Reset Zoom: Click the center focus button to reset view

Axis Range Controls

  • Auto Range: Automatically fits all data points
  • Custom Range: Set specific minimum and maximum values
  • Date/Time Support: Handles date and time variables automatically
  • Reset Button: Return to automatic scaling

Data Point Interaction

  • Hover Tooltips: View exact values by hovering over points
  • Point Highlighting: Points enlarge when hovered
  • Dataset Identification: Tooltips show which dataset each point belongs to

Statistical Analysis Features

Correlation Analysis

The insights panel provides correlation statistics:

  • Correlation Coefficient: Measures linear relationship strength (-1 to +1)
  • R-Squared Value: Percentage of variance explained by the relationship
  • Regression Equation: Mathematical formula for the trend line

Outlier Detection

  • IQR Method: Uses interquartile range to identify outliers
  • Visual Highlighting: Outliers appear in orange with larger size
  • Outlier Count: Statistics panel shows number of outliers detected

Data Quality Assessment

  • Quality Score: Overall data quality rating (0-100)
  • Completeness: Percentage of valid data points
  • Correlation Strength: Weak, moderate, or strong relationship

Interpreting Scatter Plots

Relationship Patterns

  • Positive Correlation: Points trend upward from left to right
  • Negative Correlation: Points trend downward from left to right
  • No Correlation: Points appear randomly scattered
  • Non-linear Relationships: Curved or complex patterns

Trend Line Analysis

  • Slope: Steepness indicates strength of relationship
  • R-Squared: Higher values indicate better fit
  • Intercept: Y-value when X equals zero

Comparative Analysis

  • Different Slopes: Indicates different relationship strengths
  • Parallel Lines: Similar relationships with different baselines
  • Intersecting Lines: Relationships change at different variable levels

5.3 Multi-variate Analysis

Multi-variate scatter plots allow you to analyze relationships between multiple variables simultaneously, creating comprehensive visualizations of complex data interactions.

Setting Up Multi-variate Plots

Step 1: Variable Selection

  1. Navigate to the Multi-Variate Scatter Plot tab
  2. Select multiple X (Independent) Variables using the multi-select dropdown
  3. Select multiple Y (Dependent) Variables using the multi-select dropdown
  4. The system automatically creates all possible X-Y combinations

Step 2: Pair Management

  • Active Pairs: Check/uncheck variable pairs to include in the visualization
  • Color Coding: Each pair gets a unique color for identification
  • Pair Labels: Clearly labeled as "X Variable vs Y Variable"

Step 3: Performance Optimization

For large datasets (>2000 points), the system automatically:

  • Reduces Point Size: Smaller points for better visibility
  • Adjusts Opacity: Lower opacity to prevent overplotting
  • Uses Canvas Rendering: Faster rendering for many points
  • Spatial Indexing: Efficient hover detection

Advanced Customization

Point Appearance Settings

  • Point Size: Adjustable from 4-16 pixels
  • Opacity: Control transparency from 30-100%
  • Color Override: Customize colors for specific variable pairs

Dataset Blending

  • Color Blending: Combines pair colors with dataset colors
  • Visual Distinction: Different shades for "with" vs "without" product
  • Blend Ratio: 70% pair color, 30% dataset color

Interactive Navigation

Zoom and Pan Controls

  • Zoom In/Out: Dedicated buttons for precise control
  • Reset Zoom: Return to original view
  • Mouse Interaction: Scroll to zoom, drag to pan

Axis Range Management

  • Auto-scaling: Automatically fits all data points
  • Manual Range: Set custom X and Y axis limits
  • Range Reset: Quick return to auto-scaling

Performance Features

Efficient Rendering

  • Canvas-based: High-performance rendering for large datasets
  • Grid Indexing: Fast spatial queries for hover detection
  • Throttled Updates: Smooth interaction even with many points

Memory Management

  • Lazy Loading: Only renders visible data
  • Efficient Data Structures: Optimized for large datasets
  • Garbage Collection: Automatic cleanup of unused resources

Analysis Capabilities

Pattern Recognition

  • Cluster Identification: Spot groups of similar data points
  • Relationship Comparison: Compare correlations across variable pairs
  • Outlier Detection: Identify unusual data points across multiple dimensions

Comparative Analysis

  • Cross-Variable Relationships: See how different variables interact
  • Dataset Differences: Compare patterns between conditions
  • Variable Importance: Identify which variables show strongest relationships

Export and Documentation

  • High-Resolution Export: PNG format with 2x scaling
  • Automatic Watermarking: AbhiStat branding included
  • Descriptive Filenames: Includes variable information and timestamp

5.4 Bootstrapping

Bootstrap analysis provides statistical confidence intervals for mean differences between "with product" and "without product" conditions using resampling techniques.

Understanding Bootstrap Analysis

What is Bootstrapping?

  • Resampling Method: Creates thousands of simulated samples from your data
  • Statistical Inference: Estimates confidence intervals without assumptions about data distribution
  • 10,000 Samples: AbhiStat uses 10,000 bootstrap samples for robust results
  • Mean Difference Focus: Analyzes differences between group means

Key Concepts

  • Mean Difference: Average difference between "with" and "without" product groups
  • Confidence Interval: Range of likely values for the true mean difference
  • Statistical Significance: Whether the confidence interval excludes zero
  • Bootstrap Distribution: Distribution of mean differences from resampling

Using the Bootstrap Interface

Column Selection

  1. Navigate to the Bootstrapping tab
  2. Use the Select Column for Analysis dropdown
  3. Choose any numeric variable from your dataset
  4. The analysis results update automatically

Results Overview

The summary panel shows:

  • Total columns analyzed: Number of variables processed
  • Significant impact: Variables with confidence intervals not including zero
  • No significant impact: Variables with confidence intervals including zero

Interpreting Bootstrap Results

Statistics Table

For each selected column, view:

  • Mean Difference: Average difference between groups
  • Standard Deviation: Variability of the bootstrap distribution
  • 95% Confidence Interval: Range containing the true difference with 95% confidence
  • Lower Bound (2.5%): Bottom of the confidence interval
  • Upper Bound (97.5%): Top of the confidence interval

Significance Determination

  • Significant Impact: Confidence interval does not include zero
  • No Significant Impact: Confidence interval includes zero
  • Color Coding: Red chips for significant, green chips for non-significant

Managing Results

View Options

  • Grid View: Displays variables as colored chips
  • List View: Shows variables in a table with mean differences
  • Sorting: Order by impact magnitude or alphabetically
  • Pagination: Navigate through large numbers of variables

Items Per Page

  • 10, 20, 50, or 100: Choose how many variables to display
  • Separate Pagination: Independent controls for significant and non-significant results

Export Capabilities

Excel Export

Click "Export Excel" to download comprehensive results:

  • Column Selection: Choose specific variables to include
  • Statistic Selection: Pick which statistics to export
  • Custom Filename: Set your preferred file name
  • Timestamp: Automatic date/time stamp added

PNG Export

  • Statistics Table: Download the current table as a high-resolution image
  • Watermarked: Includes AbhiStat branding
  • Print-Ready: Suitable for reports and presentations

Statistical Interpretation Guide

Confidence Interval Analysis

  • Narrow Intervals: More precise estimates, larger sample sizes
  • Wide Intervals: Less precise estimates, more variability
  • Positive Range: "With product" group has higher values
  • Negative Range: "Without product" group has higher values

Effect Size Interpretation

  • Small Effect: Mean difference less than 0.2 standard deviations
  • Medium Effect: Mean difference around 0.5 standard deviations
  • Large Effect: Mean difference greater than 0.8 standard deviations

Practical Significance

  • Statistical vs Practical: Significant results may not always be practically important
  • Context Matters: Consider the real-world meaning of the difference
  • Sample Size Effects: Large samples can detect very small differences

5.5 Advanced Visualization Features

AbhiStat provides sophisticated customization and export capabilities to create publication-ready visualizations.

Chart Settings Modal

Access advanced customization through the settings icon (⚙️) available on all visualization tabs.

Opening Chart Settings

  1. Navigate to any visualization tab
  2. Click the Settings icon in the toolbar
  3. The Chart Settings modal opens with multiple customization sections
  4. Changes are previewed in real-time

Settings Modal Features

  • Tabbed Interface: Organized sections for different customization types
  • Real-time Preview: See changes immediately
  • Reset Option: Return to default settings anytime
  • Apply/Cancel: Control when changes take effect

Point Appearance Customization

Point Size Control

  • Range: 4-16 pixels diameter
  • Interactive Slider: Real-time adjustment with visual feedback
  • Performance Optimization: Automatic size reduction for large datasets
  • Hover Effects: Points enlarge on mouse hover

Opacity Settings

  • Range: 30-100% transparency
  • Overplotting Prevention: Lower opacity helps with dense data
  • Layer Visibility: Control how overlapping points appear
  • Performance Mode: Automatic opacity reduction for >2000 points

Color Customization System

Gradient Color Picker

  • Predefined Palette: 10 carefully selected colors

  • Accessibility: High contrast ratios for readability

Variable Pair Colors

Dataset Color Coding

Export and Sharing Capabilities

High-Resolution PNG Export

Watermark System

Rendering Optimization

Zoom and Pan System

  • Drag to Pan: Click and drag to move around
  • Zoom Controls: Dedicated buttons for precise control
  • Reset Functionality: Quick return to original view

Tooltip System

  • Hover Activation: Appears on mouse hover
  • Contextual Information: Shows relevant data values
  • Dataset Identification: Indicates which group the point belongs to
  • Formatted Values: Proper number formatting and units

Grid and Axis Controls

  • Toggle Grid Lines: Show/hide background grid
  • Custom Axis Ranges: Set specific min/max values
  • Auto-scaling: Automatic range adjustment
  • Date/Time Support: Proper formatting for temporal data

Best Practices for Visualization

Color Selection Guidelines

  • High Contrast: Ensure colors are distinguishable
  • Colorblind Friendly: Avoid red-green combinations
  • Print Compatibility: Colors that work in grayscale
  • Brand Consistency: Match your organization's colors

Point Size and Opacity

  • Data Density: Smaller points for dense data
  • Visibility Balance: Ensure all points are visible
  • Overplotting Prevention: Use transparency for overlapping points
  • Focus Areas: Larger points for important data

Export Preparation

  • Clean Layout: Remove unnecessary elements
  • Clear Labels: Ensure all text is readable
  • Appropriate Size: Consider final use case
  • Quality Check: Preview before final export

Troubleshooting Visualization Issues

Performance Problems

  • Large Datasets: System automatically optimizes for >2000 points
  • Slow Rendering: Reduce point size and increase transparency
  • Memory Issues: Close other browser tabs and applications
  • Browser Compatibility: Use modern browsers for best performance

Display Issues

  • Missing Points: Check axis ranges and zoom level
  • Overlapping Labels: Adjust chart size or font settings
  • Color Problems: Verify color settings and monitor calibration
  • Export Quality: Ensure sufficient resolution for intended use

Interaction Problems

  • Zoom Not Working: Click on chart area first to focus
  • Tooltips Not Appearing: Hover directly over data points
  • Settings Not Saving: Click "Apply Changes" before closing modal
  • Reset Issues: Use the reset button to return to defaults
