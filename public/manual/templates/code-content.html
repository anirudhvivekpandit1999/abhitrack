Complex Formula Examples

Example 1: Z-Score Calculation

Calculate the z-score for standardizing data:

// Z-score formula: (value - mean) / standard_deviation
z_score = (data_column - mean(data_column)) / std(data_column)

Example 2: Percentage Change

Calculate percentage change between two values:

// Percentage change formula: ((new_value - old_value) / old_value) * 100
percentage_change = ((new_column - old_column) / old_column) * 100

Example 3: Conditional Logic

Use conditional statements in formulas:

// If-then-else logic
category = if(value > 100, "High", if(value > 50, "Medium", "Low"))

status = if(score >= 90, "Excellent", 
          if(score >= 80, "Good", 
          if(score >= 70, "Average", "Below Average")))

Reference other columns in your formulas using these methods:

Error Handling

Handle potential errors in your formulas:

// Handle division by zero
safe_division = if(denominator != 0, numerator / denominator, 0)

clean_value = if(isnull(column_name), 0, column_name)

safe_log = if(value > 0, log(value), null)

API Integration Examples

For advanced users, here are examples of API interactions:

Python Example

import requests
import json

# API endpoint
url = "https://api.abhistat.com/v1/calculate"

# Request payload
payload = {
    "session_id": "your_session_id",
    "formula": "result = column_a * 2 + column_b",
    "column_name": "calculated_result"
}

# Make request
response = requests.post(url, json=payload)

# Handle response
if response.status_code == 200:
    result = response.json()
    print("Calculation successful:", result)
else:
    print("Error:", response.text)

Best Practices:
  • Always validate your formulas before applying them to large datasets
  • Use descriptive names for calculated columns
  • Handle edge cases like null values and division by zero
  • Test formulas with sample data first
Common Errors:
  • SyntaxError: Check parentheses and operator placement
  • NameError: Verify column names are spelled correctly
  • TypeError: Ensure data types are compatible
  • ZeroDivisionError: Add checks for division by zero
